Mobil version av Webbsidan

När man gör en mobilversion av en webbsida är det viktigt att kunna använda den utan att behöva zooma ut och in en för att till exempel komma åt menyn ordentligt. Av den anledningen valde jag att ändra om menyn så den täcker hela skärmbredden och har lite högre höjd för att vara lättare att klicka på om man har lite större fingar.

Font storleken behöve också tänkas över lite annars tog till exempel rubriker upp på tok för mycket plats. Storleken på själva innehållet i övrigt tyckte jag var bra som det var så jag ändrade inte det, men det blev mer lättläst om man hade lite mer luft runt texten därav lite padding runt innehållet.

Jag tog också bort några bakgrundsbilder som inte fyllde någon egentlig funktion för att spara plats och för att inte användaren ska behöva ladda ner lika mycket filer. Bilder som tidigare hade float fick den egenskapen bortagen för att det blev lättare att läsa texten på så sätt.

Till input fälten på kontakt-sidan ligger labeln ovanför sitt tillhörande fält för att det ska få plats ordentligt och inte se konstigt ut.
